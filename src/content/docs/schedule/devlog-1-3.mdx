### DevLog 1-3 

#### Create a Node.js project and publish it on Github

This [Introduction to NPM and Node](https://omundy.github.io/learn-javascript/topics/node/slides.html) tutorial builds a foundation for what you will learn in this class. Below find the learning objectives for this DevLog which correspond to sections in the tutorial. Save the information with the ✏️ which you will enter in your DevLog in the next class.

1. Install and use Node.js on your machine
1. Confirm your Node version ✏️`v22.19.0`
1. Run a script with Node.js
1. Initialize a new project with NPM
1. Install Nodemon globally to automatically reload scripts
1. Install the Cron NPM package and use it in a script
1. [Look up](https://crontab.guru) the cron expression for "every 10 minutes". Test it and add it ✏️`*/10 * * * *` {/* https://crontab.guru/every-10-minutes */}
1. Create a Git repo and commit a `.gitignore` file
1. Publish the repo on Github. Add the link ✏️ [GitHub Repo](https://github.com/eliejerjees/cron-demo)
1. Turn it in by posting the Github link in Moodle
